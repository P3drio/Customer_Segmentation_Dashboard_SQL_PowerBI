# Online Retail Data Analysis

## Overview
This project analyzes retail transaction data to extract business insights related to customer behavior, sales trends, and revenue performance. The workflow includes data cleaning, exploratory analysis, and database integration, with results visualized in Power BI.

## Project Structure

.

├── data/
│ ├── raw/ # Original dataset
│ └── cleaned/ # Processed datasets ready for analysis
├── PowerBI/ # Power BI reports and dashboards
├── Python/ # Python scripts
├── SQL/ # SQL scripts and queries
